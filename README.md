# My AI Idea

Building AI course project

## Summary
This project proposes an AI system that automatically classifies incoming HR and payroll emails in a public administration. The AI analyzes the content of messages and routes them to the correct department, such as payroll, taxation, family situation, or administrative procedures. This would help reduce manual sorting, save time for staff, and improve response speed for citizens and employees.

## Background
Public administrations receive many emails every day. Many of them are repetitive, such as questions about salaries, documents, taxes, or family situation changes. Sorting these emails manually takes time and can slow down responses.

## How it is used
When an email arrives, the AI analyzes the text and predicts the most relevant category, for example:
- payroll
- family situation
- taxation
- administrative documents
- technical support

The email is then routed automatically to the correct service.

## Data sources
The system could be trained using:
- anonymized emails
- categories assigned by employees
- keywords and metadata from messages

Personal data would be anonymized before training the model.
